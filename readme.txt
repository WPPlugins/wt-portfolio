Управление элементами портфолио в панеле администратора

== Description ==

<b>WT Portfolio</b> - микро-плагин добавляющий сущность "Портфолио" в панель администратора и таксономию "Категории".

Если не создавать свои шаблоны для отображения элементов портфолио, они будут выводиться на базе стандартных шаблонов вывода публикаций блога (<b>archive.php</b>, <b>single.php</b>). Для настройки своего оформления для страниц "Портфолио", необходимо создать следующие шаблоны:<br />
<b>archive-portfolio.php</b> - общий вывод элементов портфолио.<br />
<b>single-portfolio.php</b> - просмотр отдельно-выбранного элемента портфолио.

Важно: Обновите ссылки ЧПУ после активации плагина.

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.

1. Найти плагин в панели администратора вашего сайта:
Плагины->Добавить новый->Поле ввода "Поиск плагинов"-> Ввести "WT Portfolio".
2. Установить найденный плагин.
3. Активировать плагин WT Service Manager:
Плагины->Установленные->Активировать WT Portfolio.
